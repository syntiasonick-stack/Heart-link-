# HeartLink Chat & Marketplace

**Owner / Developer:** Omotayo Fanegan  
**Project Type:** Web App (Chat + Marketplace)  
**Tech Stack:** HTML, CSS, JavaScript, Supabase (Auth + Database + Realtime)

---

## 🛠 Features

- **User Authentication:** Sign up and login using email/password.
- **Chat System:** Real-time chat powered by Supabase Realtime.
- **Coin System:** Users start with coins, spent to send messages or buy items.
- **Marketplace:** Users can list items for sale and purchase items with coins.
- **Admin Ready:** Optional admin panel to monitor users, messages, and items.
- **Responsive Design:** Works on desktop and mobile.
